# Huerbook

Online chatroom with Python (Flask) at the server and Javascript at the client.

## Features:
- Two factor authentication with email.
- Chat history.
- One chatroom.
- Socketio to connect JS and Python.
- Sqlite3 as database.
- Bootstrap reponsive.

### Requirements:
	flask
	flask-socketio
